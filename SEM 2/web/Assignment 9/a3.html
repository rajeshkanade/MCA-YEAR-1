Inheritance

Inheritance is a mechanism in object-oriented programming that allows one class to inherit the properties and methods of another class.

Polymorphism

Polymorphism allows methods to do different things based on the object it is acting upon, even if they share the same name.

Encapsulation

Encapsulation is the bundling of data and methods that operate on that data within one unit, such as a class in object-oriented programming.
